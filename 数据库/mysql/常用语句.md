# 数据库常用语句

## 用户赋权

```sql
-- 创建库
create database `p10-216` DEFAULT CHARACTER SET utf8 DEFAULT COLLATE utf8_general_ci;
-- 创建用户
create user 'p10-216'@'%' identified with mysql_native_password by 'pde#12345';
-- 赋权
GRANT SELECT, INSERT, UPDATE,  DELETE ON `p10-216`.* TO 'p10-216'@'%';
```
