# docker

## 容器运训空间不足

### 问题报错：

 ```bash
 library initialization failed - unable to allocate file descriptor table - out of memoryAborted (core dumped)
 ```

### 处理方案：

> 添加 `--ulimit nofile=1024:1024`参数限制大小

```shell
docker run -d -p 9006:9006 --name pdes -m 1G \
	--restart=always \
--ulimit nofile=1024:1024 \
    pde_images:10.1.0
```

# nginx

## `content-typ`相应类型不对

### 问题报错：

无

### 解决方案：

> 追加默认`content-type` mjs结尾的文件相应`content-type`为 `application/javascript`

 ```conf
location /download {
	types { 
		application/javascript mjs;
  }
  default\_type application/octet-stream;
}
 ```
